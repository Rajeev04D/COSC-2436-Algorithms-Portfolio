# COSC-2436-Algorithms-Portfolio

A semester-long collection of 12 chapter labs covering data structures and algorithms in Python. Each chapter folder contains the working program files plus a written lab report.

## Author

Rajeev Oad - COSC 2436, Spring 2026

## Description

This portfolio shows my work across the main algorithm and data structure topics studied during the semester. The chapters include searching, sorting, recursion, hash tables, graphs, trees, shortest paths, greedy algorithms, dynamic programming, and regression. My goal was to organize the labs clearly and explain each algorithm in a way that demonstrates both implementation and understanding.

| Folder | Topic |

Each folder contains the program files from that lab plus a `CHxx_LAB_REPORT.md` file documenting the implementation, test results, reflection questions, and challenges encountered.

## How to Run a Lab

Each lab is a standalone Python program. Navigate into the chapter folder and run the main file:

```bash
cd CH01
python3 main.py
```

## Requirements

- Python 3.10 or newer
- No external packages are required for the included sample programs.
- If a future lab uses additional packages, install them with:

```bash
pip install -r requirements.txt
```
